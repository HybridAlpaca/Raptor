Detailed Description

Partial-RAII wrapper class around an OpenGL shader program object.

Author
Seth Traman
Date

Deprecated:
Shader can be expected to be factored out in a near-future update. This will be accompanied by several other rendering-related changes.
Warning
This class does not free allocated GL resources on destruction, so you may want to manually track shader ID's and free them yourself. This is not expected to be implemented due to the class's limited lifespan.
